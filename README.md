# README

Resevamos challenge

REST API which can be consumed by a client app to check the weather forecast for the destinations available in Reservamos.

* Ruby version: '2.6.6'

* Rails version: '~> 6.0.6', '>= 6.0.6.1'

* Configuration
Install:

    ``bundle install``
  
Run:

    ``rails s``

Rails running in localhost:3000

* Test suite
Run:

    ``rspec``
